# Customer Personality Analysis

This project focuses on cleaning and preparing customer marketing data
to understand customer behavior and spending patterns.

## Data Cleaning Steps
- Fixed separator issue while loading data
- Handled missing values in income column
- Checked duplicates & data types
- Standardized education and marital_status text values
- Converted dt_customer to datetime
- Fixed column names to snake_case
- Created new features:
  - age
  - total_spent

## Tools Used
- Python
- Pandas
- Jupyter Notebook
